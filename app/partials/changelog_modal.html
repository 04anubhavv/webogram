Telegram recent updates

Telegram has been updated!

current version Version 0.1.3

  • Added this welcome message
  • Added keyboard shortcuts:
    • Tab - set focus to message field
    • Esc or Shift+Tab - focus search field
    • Up/Down (while in search field) - move around dialogs
    • Enter (while in search field) - open selected or first dialog
    • Alt+[0-9] - Switch to Nth dialog
    • Alt+Up/Down - move to previous/next dialog
  • Improved messages grouping
  • Fixed video modal position bug
  • Fixed mobile focus issues
